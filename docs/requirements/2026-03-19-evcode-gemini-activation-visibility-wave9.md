## Goal

Show the user how to activate Gemini within EvCode using prompts and how to observe Gemini routing/availability in runtime outputs.

## Deliverable

- A practical activation guide
- A fresh local example run proving Gemini-request routing visibility
- Governed proof and cleanup artifacts

- Explain the prompt patterns that route to Gemini.
