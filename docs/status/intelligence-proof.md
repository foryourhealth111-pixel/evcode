# EvCode Intelligence Proof

## Goal

Prove that EvCode applies governed runtime intelligence instead of relying on ad hoc prompt drift.

## Required Evidence

- intent contracts preserve goal, constraints, acceptance criteria, and assumptions
- plans preserve internal grade, wave structure, verification commands, and cleanup expectations
- execution remains traceable to requirement and plan artifacts
- benchmark mode records inferred assumptions instead of silently guessing
- host-level bridge fixtures preserve governed context injection semantics
- source-level host seam analysis proves where hook-only governance ends and where a minimal patch begins

## Pass Criteria

- requirement completeness can be reviewed from artifacts
- plan traceability can be reviewed from artifacts
- cleanup is part of the success story, not a postscript
- subagent governance remains enforceable after host materialization
- the reason for any carried host patch is inspectable from materialized upstream source
