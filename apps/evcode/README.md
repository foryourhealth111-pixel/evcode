# EvCode App

This package is the standard EvCode release channel.

Defaults:

- channel: `standard`
- mode: `interactive_governed`
- profile: `standard`

This app is intended to preserve host-native CLI behavior while embedding the shared VCO governed runtime core.

Current commands:

- `evcode`
- `evcode resume --yolo`
- `evcode assemble`
- `evcode run --task "..."`
